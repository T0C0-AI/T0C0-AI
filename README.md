## 🐧 TOCO UNIVERSE

> 토코(TOCO)는 펭귄+고양이 AI 캐릭터이자, 내가 만드는 모든 것의 구심점.
> 하나의 캐릭터를 중심으로 영상 파이프라인 → 에이전트 → 인텔리전스 → 라이프까지 확장 중.

    I --> I1["Stockmind<br/>개인 투자 두뇌"]
    I --> I2["opportunity-os<br/>AI Sales Agent"]
    I --> I3["gapminer<br/>1인 제품 기회 발굴"]
