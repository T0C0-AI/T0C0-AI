## 🎯 프로젝트 소개

**K-Studio**의 개발자 **강다니엘**이 만들고 있는 프로젝트입니다.

> 하나의 AI가 아닌, **팀으로 움직이는 AI**를 꿈꿉니다.
>
> AI 에이전트가 사람처럼 협업하고, 스스로 판단하고, 함께 문제를 해결하는 세상을 만들기 위해 
> **토코** — AI 에이전트 오케스트레이션 플랫폼을 설계 중입니다.

<br>

## 🏗️ 핵심 기능

| 기능 | 설명 |
|:----:|:---|
| **🖥️ Computer** | macOS 데스크톱 자동화 및 제어 |
| **🎙️ Voice** | 회의 전사, 실시간 다국어 동시통역 |
| **🔌 Orchestrator** | 에이전트 배포 및 작업 지시 |
| **👁️ Recognition** | 제스처 인식, 안면인식 보안 |
| **📚 RAG** | 개인 문서 인덱싱, 지식 시스템 |
| **📊 Visibility** | 에이전트 팀 모니터링 대시보드 |
| **⚡ Automation** | Notion, Slack, Discord 자동화 |
| **🧪 Recipe** | 프로젝트 분석 및 자동 레시피 생성 |
